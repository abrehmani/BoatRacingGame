# Boat Racing Game

A high-speed boat racing game developed in Unity 6 LTS using the Universal Render Pipeline (URP).

## Features
- Physics-based boat movement and controls.
- AI-controlled opponent boats with waypoint navigation.
- Checkpoint-based lap system.
- Race timer and results UI.
- Interactive environment with obstacles.
- Professional UI including Main Menu, HUD, and Pause Menu.

## Requirements
- Unity 6 LTS
- Universal Render Pipeline (URP)

## Development
This project follows the Agile Scrum methodology, divided into three sprints as outlined in the project report.

### Sprint 1: Game Design and Planning
- Defined game concept and mechanics.
- Project structure setup.

### Sprint 2: Core Mechanics
- Boat controller and physics.
- AI opponent system.
- Lap and checkpoint logic.

### Sprint 3: UI, Audio, and Polish
- HUD and Menu systems.
- Audio integration.
- Final testing and optimization.
